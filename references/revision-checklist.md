# 修改检查清单(写作/修改漏斗)

**从最大尺度到最小尺度**地修改。在确定段落能保留之前,别去打磨句子;在章节结构
定下来之前,别去打磨段落。每个层级单独走一遍。

## 动笔前的动机问题

- **降低门槛。** 空白页令人生畏;烂草稿可以修改。允许自己先写烂,再修。
- 准备好环境、留出时间、起草时让内心的批评者闭嘴。
- 把*生成*和*编辑*分开——它们用的是不同的心智模式,同时做会互相打架。

## 第 1 遍——文档尺度

- [ ] 这项科研真的值得发表吗?目标期刊/读者选对了吗?
- [ ] 全文是否围绕一个清晰主张讲一个清晰的故事?
- [ ] 整体组织是否得当(IMRaD 或有意为之的替代结构)?
- [ ] 标题是否准确、简洁地概括了贡献?

## 第 2 遍——章节尺度

- [ ] 把所有标题抽到一页纸上。单看提纲能否讲清故事?
- [ ] 各层级标题是否平行、具有描述性、简洁?
- [ ] 每个章节是否各司其职(见 `paper-structure.md`),没有重叠或空白?
- [ ] 引言:共识 → 钩子/冲突 → 回应。钩子是真实的吗?
- [ ] 摘要:目标、方法、具体结果、结论;不超期刊上限;能独立成立?
- [ ] 图/表是否按顺序被引用,且每个都站得住脚?

## 第 3 遍——段落尺度

- [ ] 一段一个主旨,由主题句开头?
- [ ] 4–8 句;没有孤立的单句成段?
- [ ] 段内连贯:重复、列举、过渡到位?
- [ ] 段间连贯:每段都与前后衔接?(只读每段的首句+尾句来检验。)

## 第 4 遍——句子尺度

- [ ] 在主动更有力之处,被动是否已改成主动?
- [ ] 软弱动词(是/有/发生/存在/被观测到)是否换成了动作动词?
- [ ] 名词化是否变回动词("比较"而非"进行比较")?
- [ ] 主谓距离短;动词没有被埋?
- [ ] 动词时态一致且恰当?
- [ ] 列表、并列、比较中的平行结构?
- [ ] 比较完整;修饰语放对位置;否定改成肯定?
- [ ] 句子的长短和节奏有变化?

## 第 5 遍——词语尺度

- [ ] 啰嗦短语已削减(见 `word-usage.md` 对照表)?
- [ ] 空洞的开头("it should be noted that…")已删?
- [ ] 每个词都精确;含糊名词(东西、方面、因素)已具体化?
- [ ] 行话和缩写适合读者,且首次出现时已定义?

## 第 6 遍——细节

- [ ] 语法、拼写、标点。
- [ ] 数字、单位、有效数字一致;无歧义日期。
- [ ] 引用格式和参考文献列表严格符合期刊要求。
- [ ] 格式遵循 Instructions to Authors(字体、页边距、行号等)。

## 编辑技巧

- **缩写式精炼(précis):** 把臃肿的段落尽量收紧地重写一遍,再把两版的精华合并——这是找出"什么是必要的"的快捷方法。
- **朗读**以捕捉节奏问题、长句失控和漏词。
- **放一放**,再以新鲜的眼睛重读;你会看出自己的注水。
- **从容接受反馈:** 读者报告的每一处困惑都是真实数据,哪怕他们提出的修法是错的。把诊断(相信它)和处方(由你定夺)分开。
- **关注小细节**——符号、大小写、连字符、术语的一致性,传递出你的用心,赢得信任。

## 署名与伦理(别跳过)

- **署名**要求实质性的智识贡献——对构思/设计、数据/分析,*以及*起草/修改——而不仅是出资或指导。尽早就署名及顺序达成一致,以免日后纠纷。
- **通讯作者**负责沟通、准确性,并确保所有列名作者都批准了最终版本。
- **绝不**捏造或篡改数据、剽窃(包括自我剽窃/重复发表)、或未经署名和许可而复用文字或图。后果足以断送职业生涯。拿不准时,就披露并引用。
